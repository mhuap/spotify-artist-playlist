# Artist Playlist Maker

Make playlist of entire artist's discography

1. Login to spotify
2. Enter artist name
3. Make playlist

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Run the client:

```bash
cd client
npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
